# Focus Flow

Мини-проект на React + Vite.

## Идея проекта

Трекер привычек и задач, в котором можно:

- просматривать список привычек;
- добавлять новые элементы;
- изменять существующие;
- удалять элементы;
- искать по названию и категории;
- фильтровать активные и выполненные.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
